1. wget https://releases.rancher.com/install-docker/20.10.sh
2. sh 20.10.sh
3. sudo systemctl status docker
4. sudo chmod 777 /var/run/docker.sock
5. git clone (მოცემული იქნება პირობაში ლინკი)
6. cd (დაკლონილი რეპო)
7. vi Dockerfile
8. docker build -t (აპლიკაციის სახელი) .
9. docker run -d -p (ჰოსტის პორტი):(კონტეინერის პორტი) (აპლიკაციის სახელი)
10. Security group-ში edit inbound rule -პორტის გახსნა, Type- Customer TCP, Port: host-ს მხარე, CDIR -0.0.0.0
11. aws ინსტანსის ინფოში ხსნი ფაბლიქ ipის https აკეთებ http ად  და ბოლოში ორი წერტილი და ჰოსტის მხარეს რა პორტიც ჩაწერე runის დროს ის
12. sudo apt install awscli
    aws configure და რამეს რანდომულად აკრეფ
    vi ~/.aws/credentials და აქ შლი რაც წერია და წერ aws credentials დეფაულტიანად
13. ქმნი ecr awsდან  ხსნი ui დან push commands
    უშვებ პირველ ქომანდს უცვლელად
    მეორე არ გჭირდება
    მესამე ქომანდში ცვლი tag ის შემდეგ რაც ეწერება იმას შენი აპლიკაციის სახელით
    მეოთხე ქომანდსაც უშვებ უცვლელად
 
 
---Dockerfile Template----------
 
FROM (პირობაში მოცემული აპლიკაციის ვერსია)
WORKDIR (პირობაში მოცემული workdir ან თუ არა მოცემული რომელი დირექტორიაც გინდა)
COPY (პირობაში იქნება მოცებული პირველ ეტაპზე რაარის გადასაკოპირებელი) ./
RUN (პირობაში მოცემული სერვისის გასაბილდი ბრძანება)
COPY . . (დიდი ალბათობით ყველაფრის კოპირება იქნება მაგრამ თუ კონკრეტული ფაილები იყო მითებული მეორე კოპირებისასაც პირველ წერტილს ჩაანაცვლებ იმ ფაილით)
EXPOSE (მითითებული პორტი)
CMD ["", ""]  ბრჭყალებში იწერება პირობაში მოცებული ბრძანებები(შეიძლება სამიც იყოს)
